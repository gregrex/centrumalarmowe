# Home to Round Audio Transitions

## Wejścia
- home -> campaign map = focus build-up
- home -> quick play = short tactical stinger
- home -> coop = anticipation with radio layer
- chapter map -> mission entry = tighter pulse
- mission entry -> lobby = social/ops bridge
- mission entry -> offline round = direct operational ramp
- mission entry -> online round = staged sync rise

## Zasady
- żadnych twardych cięć bez potrzeby,
- tła muszą crossfade'ować,
- critical UI feedback ma priorytet nad ambience.
