# Onboarding and Showcase Audio

## Potrzebne warstwy
- onboarding ambience,
- confirm / next / skip UI cues,
- coachmark ping,
- briefing underscore,
- runtime showcase cue,
- report room light stinger.

## Zasady
- onboarding ma nie męczyć,
- głos/radio ma mieć priorytet nad muzyką,
- recovery cue musi być natychmiast rozpoznawalny.
