# FULL MISSION RUNTIME AUDIO STACK

## Warstwy
1. Base city bed
2. Mission pulse
3. Dispatch feedback layer
4. Critical incident alarms
5. Objective stingers
6. Role-specific radio chatter

## Priorytety
1. mission complete gate
2. critical route failure
3. dispatch outcome fail
4. dispatch outcome success
5. objective complete
6. ambient

## Reguły miksu
- krytyczny alarm obniża ambience o 5 dB
- success stinger nie zatrzymuje gameplayu
- fail stinger ma krótki tail, bez melodramatu
- role radio chatter ma niski priorytet gdy otwarte są modale
