# SFX Event Matrix

## Krytyczne
- nowy incydent czerwony -> krótki alarm 3-tonowy + wibracja,
- utrata roli gracza -> ostrzejszy alert systemowy,
- przejęcie przez bota -> miękki alert informacyjny,
- blackout -> niski puls + szum zakłóceń,
- zdarzenie masowe -> warstwa muzyczna crisis.

## Informacyjne
- nowy średni incydent,
- przyjazd jednostki,
- domknięcie incydentu,
- wzrost obciążenia systemu.
