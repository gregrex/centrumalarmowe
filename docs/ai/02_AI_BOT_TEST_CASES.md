# Test cases dla AI botów

- Gracz Operator znika na 20 sekund podczas 3 równoległych zgłoszeń.
- DispatchBot przejmuje rolę przy braku wolnych karetek.
- OpsBot ma konflikt dwóch krytycznych zdarzeń i musi pingować człowieka.
- CrisisBot aktywuje procedurę blackout bez nadmiernej eskalacji.
- Człowiek wraca w trakcie działania bota i odzyskuje rolę bez desynchronizacji.
