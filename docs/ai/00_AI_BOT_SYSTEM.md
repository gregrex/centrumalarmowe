# System AI botów

## Cel
Zastąpić brakującego gracza lub wspomagać solo tak, aby sesja pozostawała grywalna.

## Warstwy
1. Decision Layer
   - priorytety,
   - proste klasyfikacje,
   - wybór bezpiecznych akcji.
2. Tactical Layer
   - dobór jednostek,
   - routing,
   - monitoring aktywnych zdarzeń.
3. Communication Layer
   - komunikaty do graczy,
   - oznaczanie niepewności,
   - prośba o interwencję człowieka.
4. Recovery Layer
   - oddanie roli po powrocie gracza,
   - synchronizacja kontekstu.

## Typy botów
- CallBot
- DispatchBot
- OpsBot
- CrisisBot

## Wymagania jakościowe
- przewidywalność,
- bezpieczeństwo decyzji,
- ograniczona agresywność optymalizacji,
- wyjaśnialność,
- brak spamowania.
