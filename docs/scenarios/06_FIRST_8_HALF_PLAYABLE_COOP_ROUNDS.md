# Pierwsze 8 rund półgrywalnych coop

1. Ambulans + omdlenie w centrum
2. Pożar mieszkania + korek na dojeździe
3. Bójka stadionowa + brak wolnej policji
4. Karambol + podział dwóch karetek
5. Alarm czujnika + prawdziwy pożar w drugim miejscu
6. Fala zasłabnięć w upale
7. Blackout dzielnicy + przeciążone trasy
8. Nocny overload z jednym graczem i trzema botami

Każda runda powinna mieć:
- startowy snapshot,
- 2–4 delty,
- minimum 1 shared action,
- minimum 1 decyzję o ETA/cooldown,
- partial report.
