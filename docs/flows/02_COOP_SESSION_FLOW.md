# Flow sesji coop

1. Host tworzy lobby.
2. Wybiera scenariusz i poziom trudności.
3. Gracze dołączają przez kod sesji.
4. Każdy wybiera rolę lub dostaje auto-przypisanie.
5. Puste sloty wypełnia AI.
6. Serwer tworzy snapshot startowy.
7. Countdown.
8. Active session.
9. Co tick: director, boty, synchronizacja delta.
10. Rozłączenie gracza -> grace period -> takeover.
11. Powrót gracza -> resync -> handback.
12. Koniec sesji -> raport -> zapis statystyk.
