# Główne przepływy

## Nowa gra
menu -> wybór trybu -> profil -> rola -> briefing -> start sesji

## Coop
menu -> lobby -> zaproszenie / matchmaking -> wybór ról -> readiness -> start

## Solo z botami
menu -> wybór scenariusza -> obsadzenie brakujących ról botami -> start

## Zdarzenie
zgłoszenie -> klasyfikacja -> priorytet -> przydział służb -> monitoring -> rezultat

## Eskalacja
zdarzenie aktywne -> nowa informacja -> wzrost pilności -> dodatkowe siły -> aktualizacja KPI

## Kryzys miejski
warunek director/scenariusz -> globalny alert -> zmiana presetu miasta -> aktywacja procedur specjalnych

## Reconnect / bot takeover
gracz traci połączenie -> bot przejmuje rolę -> gracz wraca -> człowiek odzyskuje sterowanie
