# Android Preview Telemetry API

## Endpointy
- GET /api/android-preview-build/demo
- GET /api/release-readiness-checklist/demo
- GET /api/telemetry-dashboard/demo
- GET /api/review-feedback-dashboard/demo
- GET /api/final-capture-pack/demo

## Zasady
- payloady demo są deterministyczne,
- nadają się do zasilenia mock UI,
- zawierają statusy, KPI i listy kontrolne,
- mogą być użyte w capture mode.
