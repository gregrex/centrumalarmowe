# Campaign Entry, Profile and Theme API

## Endpointy v11
- `GET /api/campaign-chapters/demo`
- `GET /api/mission-entry/demo?missionId=...`
- `GET /api/profile-cosmetics/demo`
- `GET /api/player-identity/demo`
- `GET /api/home-to-round-audio`

## Kontrakty
### CampaignChapterDto
Opisuje chapter, jego theme, progress i listę node'ów.

### CampaignMissionEntryDto
Opisuje mission entry sheet wraz z bot fill summary.

### ProfileCosmeticDto
Opisuje kosmetyczny element z rarity, category i unlock source.

### PlayerIdentityDto
Opisuje aktualny profil gracza i wybrane kosmetyki.

### HomeToRoundAudioDto
Opisuje routing muzyki i stingerów od home do round start.
