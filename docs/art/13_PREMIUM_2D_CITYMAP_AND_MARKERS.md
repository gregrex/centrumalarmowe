# Premium 2D City Map and Markers

## Visual target
A premium mobile 2D command interface:
- crisp map layers,
- strong marker silhouettes,
- low visual noise,
- immediate understanding of urgency.

## City map art rules
- use layered districts, roads and landmarks,
- keep roads readable at small zoom,
- avoid dense micro-detail,
- use a darker base map with bright operational overlays,
- use subtle gradients, not noisy textures.

## Marker classes
- medical,
- fire,
- police,
- mixed response,
- infrastructure issue,
- critical incident,
- bot-controlled slot marker.

## Marker readability
Each marker must work in:
- 32 px,
- 40 px,
- 64 px,
with no text embedded.

## Effects
- selected incident pulse,
- route line glow,
- critical zone heat pulse,
- soft pressure shimmer,
- dispatch confirmation flash.

## Production-friendly rule
Prefer SVG-first placeholders and atlas-ready exports over complex painterly assets.
The game should look premium because of clarity and consistency, not asset count alone.
