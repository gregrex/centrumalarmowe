# Animated Menu Day/Night/Weather Renderer

## Cel
Menu ma być żywe i premium, ale łatwe do produkcji:
- 2D layered renderer,
- parallax,
- color grading tokens,
- overlay pogodowy,
- mała liczba atlasów,
- możliwość pracy na placeholderach.

## Warstwy
1. far background skyline
2. mid city silhouette
3. roads / key buildings
4. emergency lights layer
5. UI-safe motion layer
6. weather overlay
7. vignette / glow

## Presety
- day clear,
- evening amber,
- night calm,
- heavy rain,
- storm blackout,
- winter fog.

## Zasady ruchu
- mały, spokojny ruch,
- brak agresywnego motion sickness,
- reduced motion = tylko crossfade i lekki light pulse.
