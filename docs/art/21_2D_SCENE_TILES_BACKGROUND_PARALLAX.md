# 2D Scene Tiles, Backgrounds, Parallax

## Warstwy
- Far background: skyline / clouds / weather
- Mid background: city blocks / bridges / district landmarks
- Near background: road, bay, console frame, props

## Tiles
- road straight,
- road junction,
- bridge section,
- tram track,
- district edge,
- industrial lot,
- hospital lot,
- emergency bay,
- command center interior frame.

## Parallax rules
- Far: 0.15 speed
- Mid: 0.35 speed
- Near: 0.65 speed
- UI foreground: static

## Dla mobile
- nie używaj zbyt wielu przezroczystości,
- utrzymuj spójne kadrowanie,
- ogranicz ilość animowanych warstw do 2–3.
