# Mobile 2D Readability — Golden Rules

1. Never let art reduce decision speed.
2. The player must find the active incident in under one second.
3. Critical CTA must stay in the lower thumb zone.
4. Top bar must remain stable; avoid layout jumping.
5. Use color plus shape, never color alone.
6. One expanded card at a time.
7. Marker silhouettes must differ by service and severity.
8. Text blocks must stay short and scannable.
9. Pressure and time must be visible without opening extra screens.
10. Route lines only for selected items.
11. Keep low-frequency information collapsed by default.
12. Use bottom sheets for action, not for dense reports.
13. Report timeline can be denser than live HUD.
14. Missing human players must be visible with clear AI labels.
15. Art polish is welcome only if it does not add clutter.
