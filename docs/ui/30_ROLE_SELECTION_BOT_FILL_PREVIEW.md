# Role Selection i Bot Fill Preview

## Role
- operator
- dispatcher
- coordinator
- crisis_officer

## Widok
Każda rola ma własną kartę zawierającą:
- nazwę,
- krótki opis,
- poziom trudności,
- recommended tag,
- slot state,
- kto zajmuje slot,
- jaki bot przejmie rolę jeśli slot pusty,
- bonus i ryzyka dla tej roli.

## Stany slotów
- open
- reserved
- locked_by_mode
- bot_assigned
- player_assigned
- reconnect_pending
- unavailable

## Bot fill preview
Panel pokazuje:
- przewidywany skład drużyny,
- profile botów,
- styl zachowania,
- expected communication frequency,
- weak spots AI.
