# Review build checklist and capture mode

## Ekrany objęte review
- Splash
- Home
- Chapter Map
- Mission Entry
- Briefing
- Runtime HUD
- Recovery Cards
- Scoreboard
- Reward Reveal
- Retry / Next

## Capture mode
Capture mode wyłącza:
- debug labels,
- zbędne identyfikatory techniczne,
- testowe watermarki,
- nadmiarowe hinty,
- utility overlaye niezwiązane z pokazem.

Capture mode zachowuje:
- krytyczne CTA,
- najważniejsze alerty,
- dynamiczne trasy,
- unit markers,
- objective tracker.

## Checklist UI
- brak obciętych elementów na 9:16,
- spójne marginesy i safe area,
- czytelne CTA primary/secondary,
- brak pustych kart i pustych paneli,
- czytelne teksty fail/partial/success.
