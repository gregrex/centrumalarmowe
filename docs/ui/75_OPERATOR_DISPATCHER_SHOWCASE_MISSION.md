# Operator + Dispatcher showcase mission

## Cel demonstracyjny
Jedna misja ma pokazać współpracę dwóch najważniejszych ról:
- operator odbiera i klasyfikuje zgłoszenie,
- dispatcher wysyła i koryguje jednostki.

## Minimalny loop
1. przyjęcie zgłoszenia,
2. nadanie priorytetu,
3. wybór jednostki,
4. dispatch,
5. obserwacja trasy,
6. eskalacja lub recovery,
7. wynik i raport.

## Dlaczego ten wariant
To najkrótsza ścieżka do grywalnego, zrozumiałego i filmowalnego vertical slice.
