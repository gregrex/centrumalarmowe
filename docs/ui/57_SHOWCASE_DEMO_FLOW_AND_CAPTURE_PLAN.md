# Showcase Demo Flow and Capture Plan

## Flow pokazu
Home → Intro → Chapter Map → Mission Entry → Briefing → Ready Check → Runtime → Recovery Card → Scoreboard → Reward Reveal → Retry/Next.

## Capture beats
- Beat 01: logo + menu scena.
- Beat 02: onboarding role cards.
- Beat 03: mission briefing.
- Beat 04: dispatch na mapie.
- Beat 05: escalation + recovery card.
- Beat 06: success gate.
- Beat 07: scoreboard.
- Beat 08: reward reveal i next mission teaser.

## Ważne
Nagrywanie musi być możliwe bez debugowych overlayów. Demo mode powinien mieć stałe czasy, żeby agent mógł uzyskać powtarzalne nagrania.
