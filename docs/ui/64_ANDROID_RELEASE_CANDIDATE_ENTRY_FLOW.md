# Android RC entry flow

## Wejście gracza
1. Otwórz build RC.
2. Przejdź onboarding lub pomiń po pierwszym uruchomieniu.
3. Wejdź do showcase mission.
4. Zagraj misję solo / bot fill / coop debug.
5. Zobacz report room i reward reveal.
6. Wróć do Home lub użyj Retry/Next.

## Cele
- szybki start,
- niskie tarcie wejścia,
- zero pustych ekranów,
- pełna czytelność UI w pionie.
