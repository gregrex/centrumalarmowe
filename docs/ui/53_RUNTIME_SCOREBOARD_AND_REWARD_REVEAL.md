# Runtime Scoreboard and Reward Reveal

## Widoki
- compact scoreboard po zatrzymaniu timera
- expanded scoreboard z zakładkami: Performance / Mistakes / Rewards
- reward reveal sequence: stars -> xp -> badge -> unlock teaser

## Pola
- mission score
- stars earned
- response time median
- incidents resolved
- incidents failed
- city pressure ending state
- bonus objective result
- top recovery decision impact

## Edge cases
- fail state bez rewardu głównego, tylko minimalny xp
- partial z odblokowaniem retry hint
- success perfect z animacją premium
