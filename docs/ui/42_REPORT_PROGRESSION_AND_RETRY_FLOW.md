# Report Progression And Retry Flow

## Raport po misji
Sekcje:
1. grade summary,
2. objectives breakdown,
3. dispatch accuracy,
4. city stability delta,
5. rewards,
6. next node / retry / back home.

## Retry flow
Retry powinien:
- zachować ten sam mission seed lub pozwolić na variant,
- pokazać 3 największe błędy,
- skrócić drogę do ponownego startu.
