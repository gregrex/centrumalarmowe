# Visual Runtime Route Layer and Recovery Cards

## Route Layer
Centralna warstwa 2D na mapie w trakcie misji.

### Elementy
- segment trasy,
- punkt startu jednostki,
- punkt celu,
- aktywne warningi,
- severity badge,
- ETA label,
- state color,
- pulse highlight.

### Stany
- clear
- delayed
- blocked
- rerouted
- critical

## Recovery Decision Cards
Karty uruchamiane, gdy system wymaga decyzji odzyskującej kontrolę.

### Typy kart
- reroute traffic corridor,
- request mutual aid,
- reassign ambulance,
- delay lower priority call,
- open temporary response lane,
- assign bot operator assist.

### Zawartość karty
- tytuł,
- krótki opis problemu,
- 2-3 opcje,
- koszt ryzyka,
- wpływ na objective,
- wpływ na city pressure,
- feedback po zatwierdzeniu.
