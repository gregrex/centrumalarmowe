# Report Room Polish and Demo Final Flow

## Report Room
Pół-statyczna scena 2D po misji z:
- stolikiem raportowym,
- ścianą monitorów,
- mapą miasta w tle,
- reward reveal,
- statusem miasta,
- gwiazdkami i odznakami.
