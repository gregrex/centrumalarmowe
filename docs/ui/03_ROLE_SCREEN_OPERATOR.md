# Ekran roli — Operator zgłoszeń

## Cel
Szybkie odebranie połączenia, wydobycie informacji i sklasyfikowanie zdarzenia.

## Sekcje UI
- aktywne połączenie,
- formularz pytań dynamicznych,
- streszczenie zgłoszenia,
- sugestia kategorii AI,
- przycisk eskalacji do koordynatora,
- historia ostatnich połączeń,
- panel alertów krytycznych.

## Najważniejsze akcje
- oznacz kategorię,
- ustaw priorytet,
- potwierdź lokalizację,
- dodaj tag ryzyka,
- przekaż do dispatch,
- odrzuć jako fałszywy alarm,
- oznacz niepełne dane.
