# Menu Widgets, Cards and Transitions

## Widgety
- Continue Session
- Quick Play
- Campaign Progress
- Coop Party
- Daily Challenge
- Reward Inbox
- Scene Theme Selector
- News / patch notes
- Audio test button

## Karty
Każda karta zawiera:
- title,
- subtitle,
- icon,
- urgency badge,
- optional progress bar,
- CTA.

## Przejścia
- card expand,
- bottom sheet slide,
- overlay fade,
- parallax background shift,
- audio cue confirm.

## Zasady
- transition <= 250 ms,
- critical CTA zawsze w zasięgu kciuka,
- mała liczba warstw jednocześnie,
- bez ciężkich pełnoekranowych animacji w trybie reduced motion.
