# Store shots, capsules and showcase cards

## Typy assetów store
- main capsule,
- portrait mobile hero shot,
- dispatch action shot,
- recovery card shot,
- report room reward shot,
- next mission teaser shot.

## Zasady kompozycji
- jeden główny punkt skupienia,
- nie więcej niż 1-2 tekstowe callouty,
- brak chaosu ikon,
- postacie i pojazdy tylko jako czytelne hero obiekty,
- runtime route pokazany prostą, wyraźną linią.

## Showcase cards
Karty do wykorzystania w landing/demo:
- "koordynuj służby",
- "graj solo lub z bot fill",
- "reaguj na eskalacje",
- "naprawiaj błędy i ratuj misję",
- "przejdź od zgłoszenia do raportu".
