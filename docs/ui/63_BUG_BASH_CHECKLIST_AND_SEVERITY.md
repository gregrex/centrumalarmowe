# Bug bash checklist i severity

## Severity
- P0: crash, czarny ekran, martwy flow krytyczny.
- P1: błędne CTA, brak danych, nieczytelność blokująca misję.
- P2: błędy wizualne, zły spacing, niewłaściwe audio route.
- P3: drobne polish issues, copy, timing transitions.

## Obszary bug bash
- Home i onboarding.
- Mission entry i bot fill.
- Runtime HUD i dispatch.
- Recovery / fail / retry / next.
- Report room i reward reveal.
- Audio/music continuity.
- Capture mode i screenshots.
