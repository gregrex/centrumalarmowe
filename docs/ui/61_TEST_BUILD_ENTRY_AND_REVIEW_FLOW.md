# Test build entry and review flow

## Flow reviewera
1. Uruchom build.
2. Przejdź przez Home.
3. Wejdź do showcase mission.
4. Zobacz briefing i role.
5. Uruchom runtime.
6. Użyj recovery card.
7. Zobacz raport.
8. Obejrzyj retry / next handoff.
9. Wypełnij checklistę review.

## Flow nagrania demo
1. Włącz capture mode.
2. Załaduj showcase seed.
3. Przejdź przez zdefiniowaną ścieżkę demonstracyjną.
4. Zachowaj 3 obowiązkowe ujęcia runtime.
5. Zachowaj 1 ujęcie report room.
6. Zachowaj 1 ujęcie retry/next.
