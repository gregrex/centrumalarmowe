# Main Menu, Meta Progression and Scene Flow

## Główne ekrany
1. Splash
2. Legal / permissions
3. Main Menu
4. Profile
5. Loadout
6. Settings
7. Accessibility
8. Quick Play entry
9. Campaign entry
10. Coop Lobby entry
11. Rewards / post-session reveal

## Main Menu layout
- Top bar: profile level, currency, notifications
- Hero center card: Continue / Quick Play
- Secondary action row: Campaign, Coop, Daily, Profile
- Scene background: aktywny theme pack
- Bottom nav: Home, Play, Squad, Rewards, Settings

## Flow
Main Menu -> Quick Play -> Lobby -> Session
Main Menu -> Campaign -> Mission Select
Main Menu -> Coop -> Lobby / Invite / Bot Fill
Main Menu -> Profile -> Loadout -> Save
Main Menu -> Settings -> Accessibility / Audio / Controls

## Meta progression layer
- Po każdej rundzie gracz widzi:
  - XP,
  - role mastery,
  - unlock token,
  - cosmetic progress,
  - challenge progress.

## Design rule
Main Menu nie może wyglądać jak sklep. Ma wyglądać jak operacyjne centrum gotowości.
