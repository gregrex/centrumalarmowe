# RC player flow

## Flow docelowy
Home → Campaign Showcase → Mission Entry → Briefing → Ready Check → Runtime →
Recovery / Retry / Complete Gate → Report Room → Reward Reveal → Retry / Next Mission

## Wymagania UX
- Maksymalnie 2 tapnięcia do wejścia w showcase mission.
- Jasne CTA na wszystkich ekranach krytycznych.
- Czytelny układ w pionie 9:16.
- Spójne tło, kolory alertów, badge i font scale.
