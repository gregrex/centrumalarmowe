# Ekran roli — Dyspozytor jednostek

## Cel
Przydział najbliższych i najwłaściwszych jednostek do aktywnych incydentów.

## Sekcje UI
- mapa miasta,
- lista aktywnych incydentów,
- lista jednostek i ich statusów,
- szybkie filtry: wolne / w drodze / zajęte / niedostępne,
- panel rekomendacji AI,
- przycisk wsparcia międzydzielnicowego.
