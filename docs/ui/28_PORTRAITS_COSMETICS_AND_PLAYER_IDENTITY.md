# Portraits, Cosmetics and Player Identity

## Cel
Profil ma być lekki produkcyjnie, ale dawać poczucie własności konta.

## Elementy tożsamości gracza
- avatar / portrait,
- nameplate,
- badge,
- title,
- preferred role marker,
- frame quality,
- voice style tag (czysto kosmetyczny),
- chapter banner.

## Zasady
- kosmetyki nie wpływają na balans,
- profile muszą być czytelne na telefonie,
- portrety nie mogą być przesadnie realistyczne,
- frame + badge + title muszą mieć prosty system rzadkości.

## Rzadkości
- common,
- uncommon,
- rare,
- heroic,
- event.

## Źródła odblokowania
- campaign progress,
- daily challenge,
- coop milestone,
- flawless dispatch,
- event chapter reward.
