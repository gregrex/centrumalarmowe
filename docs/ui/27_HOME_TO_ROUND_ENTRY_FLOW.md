# Home to Round Entry Flow

## Główna ścieżka
1. Home
2. wybór Campaign / Quick Play / Coop
3. jeśli Campaign -> Chapter Map
4. wybór misji
5. Mission Entry Sheet
6. wybór trybu:
   - Offline Solo
   - Coop Online
   - Coop With Bot Fill
7. rezerwacja ról
8. uzupełnienie botów
9. loading / round bootstrap
10. wejście do HUD rundy

## Reguły
- użytkownik nigdy nie powinien wykonać więcej niż 3 krytyczne tapnięcia od Home do Start Round,
- przy braku współgraczy system domyślnie proponuje bot fill,
- tryb offline nie pokazuje konfliktów sieciowych,
- tryb online pokazuje ready state i role reservation.

## Mikro-feedback
- role reserved: krótki stinger + badge,
- bot assigned: miękki confirm + bot chip,
- mission locked: krótkie reject feedback,
- loading into round: przejście audio + weather fade.
