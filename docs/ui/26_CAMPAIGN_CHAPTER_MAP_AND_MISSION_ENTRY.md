# Campaign Chapter Map and Mission Entry

## Cel
Ekran mapy rozdziałów ma być pomostem między atrakcyjnym menu mobilnym a właściwą rundą.

## Ekran 1 - Chapter Map
### Główne komponenty
- header z profilem i bieżącym chapter theme,
- pasek postępu kampanii,
- przewijalna mapa rozdziału,
- węzły misji,
- panel rozdziału po tapnięciu,
- quick actions:
  - start latest,
  - replay,
  - details,
  - rewards,
- bottom CTA bar.

### Stany węzła
- locked,
- available,
- active,
- completed,
- challenge,
- perfect clear,
- daily boosted.

### Touch UX
- tap = select node,
- double tap = quick details,
- hold = mission comparison,
- swipe = scroll map.

## Ekran 2 - Mission Entry Sheet
### Sekcje
- tytuł,
- chapter / mission id,
- estimated length,
- difficulty,
- recommended role,
- weather + time-of-day,
- starting units,
- top 3 risks,
- rewards,
- available player slots,
- bot fill summary.

### CTA
- Play Solo
- Play With Bots
- Create Coop Lobby
- Continue Last Setup

## Edge cases
- brak unlocka,
- chwilowy brak sieci,
- mismatch wersji contentu,
- brak zapisanej preferowanej roli,
- gracz wraca z niedokończonego lobby.
