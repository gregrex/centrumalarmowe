# UI 43 — Live Route Renderer and Round Timer

## Komponenty
- top timer bar,
- route overlay layer,
- active route chips,
- ETA badges,
- current phase label,
- city pressure badge,
- mission objective micro-cards.

## Edge cases
- brak trasy -> fallback route icon + blocked state,
- dwie trasy do jednego incydentu -> wybór primary / secondary,
- timer = 0 -> mission complete gate lub fail branch.
