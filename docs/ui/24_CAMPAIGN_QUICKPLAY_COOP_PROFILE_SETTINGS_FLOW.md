# Flow: campaign / quick play / coop / profile / settings

## Home -> Campaign
1. Tap Campaign.
2. Otwórz campaign overview.
3. Pokaż aktualny node, progress, rewards.
4. Pozwól uruchomić misję lub wrócić.

## Home -> Quick Play
1. Tap Quick Play.
2. Pokaż preferred role, session preset, difficulty.
3. Start local / online / bot fill.
4. Przejście do lobby lub od razu do rundy.

## Home -> Coop
1. Tap Coop.
2. Pokaż party status.
3. Host / Join / Invite.
4. Ready check.
5. Start lobby.

## Home -> Profile
1. Tap Profile.
2. Pokazuj badges, avatar, unlocked cosmetics.
3. Wybór aktywnego loadoutu.
4. Podgląd ulubionej roli.

## Home -> Settings
1. Tap Settings.
2. Tabs:
   - audio
   - accessibility
   - controls
   - notifications
   - data/privacy
