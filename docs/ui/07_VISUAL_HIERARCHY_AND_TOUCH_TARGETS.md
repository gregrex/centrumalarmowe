# Visual hierarchy and touch targets

## Mobilne zasady
- target dotykowy minimum 44x44 dp,
- krytyczne CTA minimum 52x52 dp,
- nie więcej niż 5 krytycznych akcji w jednym panelu,
- ważność budowana wielkością, kolorem i ruchem, nie samym tekstem.

## Hierarchia
1. alarm globalny,
2. aktywny incydent wysokiego priorytetu,
3. stan jednostek,
4. akcje roli,
5. filtry i raporty.

## Wersje HUD
- compact,
- standard,
- accessibility large,
- streamer/demo.
