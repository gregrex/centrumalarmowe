# Player-Facing Polish Pack

## Obszary polishu
- płynność przejść,
- czytelne ikonografie i badge,
- animacje hover/tap/confirm,
- lepsze placeholdery portretów i kart,
- screen-specific audio cues,
- spójność koloru alertów od menu do report room.

## Must-have dla demo
- menu scene z ruchem tła,
- onboarding widgets z miękkimi wejściami,
- runtime HUD z czytelną hierarchią,
- reward reveal z krótkimi stingerami,
- retry/next z jasnym CTA.
