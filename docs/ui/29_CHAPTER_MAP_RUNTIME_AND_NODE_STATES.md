# Chapter Map Runtime i stany node'ów

## Ekran
Interaktywny chapter map z aktywnymi node'ami misji.

## Stany node'ów
- locked
- available
- selected
- active
- completed_one_star
- completed_two_stars
- completed_three_stars
- challenge
- elite
- story_gate

## Elementy UI
- chapter background
- path lines
- mission nodes
- node glow
- lock icon
- star badges
- progress rail
- objective summary panel
- recommended role banner
- weather/time strip

## Zasady czytelności
- aktywny node zawsze kontrastuje z tłem,
- completed nodes są stonowane, ale czytelne,
- challenge i elite wymagają osobnego shape language,
- nie używać małych tap targetów pod 48 px.
