# First-Time User Experience and Onboarding

## Cel
Pierwsze 2 minuty mają wyjaśnić:
- czym jest gra,
- co robi gracz,
- co oznaczają role,
- jak wygląda pojedyncza misja,
- jak wrócić do pomocy później.

## Ekrany
1. Intro carousel.
2. Role explainer.
3. Home z coachmarkami.
4. Mission entry z wyjaśnieniem `bot fill`.
5. Runtime HUD overlay.

## Reguły UX
- max 1 główny komunikat na ekran,
- duże strefy dotykowe,
- możliwość pominięcia,
- możliwość powrotu z menu pomocy,
- każda wskazówka ma przykład, nie tylko opis.
