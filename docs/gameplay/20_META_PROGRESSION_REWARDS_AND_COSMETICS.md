# Meta Progression, Rewards and Cosmetics

## Rewards
- XP
- mastery stars
- profile badges
- theme unlocks
- portrait unlocks
- UI skin unlocks
- title ribbons

## Cosmetics
Kosmetyka nie może utrudniać czytelności. Dozwolone:
- ramki portretów,
- warianty tła menu,
- subtelne skórki kart,
- znaczki roli.

## Gameplay impact
Meta progression nie może dawać pay-to-win. Dozwolone:
- convenience unlocks,
- cosmetic variety,
- challenge access,
- profile expression.
