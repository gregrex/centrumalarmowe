# Real Android build flow + bugfix freeze

## Flow
Home -> Briefing -> Ready -> Runtime -> Recovery/Fail -> Report -> Retry/Next

## Freeze priority
- P0 blocker crash/input/network freeze,
- P1 unreadable HUD / broken progression,
- P2 audiovisual polish,
- P3 nice-to-have copy tweaks.

## Reguła
Każdy nowy fix musi wskazać, czy dotyczy operatora, dispatchera, misji czy builda.
