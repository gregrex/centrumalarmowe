# Showcase Mission Script and Onboarding

## Misja pokazowa
Misja ma prowadzić gracza przez trzy kluczowe zachowania:
1. odczyt sytuacji,
2. dispatch,
3. recovery decision.

## Onboarding beats
- wyjaśnij mapę,
- wyjaśnij jednostki,
- wyjaśnij statusy celu,
- wymuś pierwszy dispatch,
- pokaż recovery card,
- pokaż scoreboard.
