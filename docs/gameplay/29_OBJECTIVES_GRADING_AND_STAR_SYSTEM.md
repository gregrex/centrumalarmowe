# Objectives, grading i star system

## Gwiazdki
- 1 star: misja ukończona,
- 2 stars: utrzymane SLA i ograniczone eskalacje,
- 3 stars: wynik wzorcowy + bonusowy objective.

## Typy objective
- response_time
- correct_dispatch
- escalation_control
- role_coordination
- bot_efficiency
- resource_preservation

## Grading
A, B, C, D, F + stars + reward preview.
