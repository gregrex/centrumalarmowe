# First Playable Mission Flow End To End

## Flow
Home -> Campaign Node -> Briefing -> Team Readiness -> Runtime Map -> 3 incydenty -> 1 escalation chain -> Mission Complete Gate -> Report -> Rewards -> Retry or Continue.

## Minimalna zawartość misji
- 1 medical critical,
- 1 traffic collision,
- 1 fire incident,
- 1 chain event triggered by delay,
- 3 objectives,
- 1 końcowy grading.
