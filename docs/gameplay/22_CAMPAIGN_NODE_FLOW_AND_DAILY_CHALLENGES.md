# Campaign node flow and daily challenges

## Campaign
Mapa kampanii ma być lekka i szybka:
- rozdziały,
- nody misji,
- nody challenge,
- nody tutorialowe,
- reward nodes.

## Daily challenge
Codzienny challenge ma:
- seed,
- ograniczenie czasu,
- preferowaną rolę,
- bonus reward,
- leaderboard-ready ID.

## Cel
Dać graczowi powód do krótkiego powrotu codziennie bez rozpychania głównego scope.
