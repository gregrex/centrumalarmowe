# Premission Briefing + Ready Check

## Flow
1. mission selected
2. briefing loaded
3. player reads objectives
4. player enters readiness
5. slots synchronize
6. bots fill missing roles
7. all ready check
8. bootstrap confirm

## Edge cases
- one player disconnects during readiness,
- host cancels,
- role lock conflict,
- bot preview stale,
- offline mode forced by network drop.
