# Quasi-Production Demo Flow — One Mission

## Misja demo
Bridge collision + smoke + blocked traffic lane.

## Loop
1. Briefing.
2. Readiness.
3. Runtime start.
4. First dispatch.
5. Route delay.
6. Recovery decision.
7. Objective transition.
8. Resolution or fail.
9. Report room.
10. Retry/Next.

## Wymogi
- czas sesji 6-10 minut,
- wysoka czytelność,
- 1 recovery branch obowiązkowo,
- 1 fail branch obowiązkowo.
