# Core loop

1. Odbierz zgłoszenie.
2. Wydobądź kluczowe informacje.
3. Nadaj kategorię i priorytet.
4. Wyślij jednostki.
5. Monitoruj status zdarzenia.
6. Reaguj na eskalację.
7. Przekieruj zasoby.
8. Zamknij zdarzenie.
9. Zaktualizuj KPI.
10. Przejdź do kolejnych zgłoszeń.

## Meta loop
- raport po zmianie,
- rozwój centrum,
- odblokowanie procedur,
- rozwój AI botów,
- zakup ulepszeń,
- odblokowanie nowych scenariuszy i mutatorów.
