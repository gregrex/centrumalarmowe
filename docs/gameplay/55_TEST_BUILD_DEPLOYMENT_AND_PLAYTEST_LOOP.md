# Test build deployment and playtest loop

## Pętla
1. Zbuduj review build.
2. Udostępnij testerowi.
3. Zbierz checklistę.
4. Zbierz screenshoty błędów.
5. Zapisz telemetry notes.
6. Uzupełnij backlog poprawek.
7. Wydaj następną iterację.

## Playtest questions
- Czy onboarding jest wystarczający?
- Czy wiadomo, co kliknąć jako następne?
- Czy recovery card była zrozumiała?
- Czy raport końcowy wyjaśnia porażkę/sukces?
- Czy UI jest czytelne na telefonie?
