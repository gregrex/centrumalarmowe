# Player Onboarding and Hint System

## Typy hintów
- persistent,
- contextual,
- corrective,
- celebratory,
- retry coaching.

## Reguły
- hint nie może blokować akcji dłużej niż 3 sekundy,
- corrective hint pojawia się tylko po realnym błędzie lub opóźnieniu,
- retry coaching bierze dane ze scoreboardu i recommendation engine.
