# V23 Android Preview + Release Readiness Scope

## Cel
Doprowadzić showcase misji do stanu **Android Preview Build**, który można:
- zainstalować jako testowy build,
- obejrzeć jako pionowy slice jednej misji,
- nagrać do trailera / store preview,
- zrecenzować przez QA, design, audio i product,
- zasilić telemetryką i feedbackiem playtesterów.

## Zakres v23
1. Android preview build pack.
2. Release readiness checklist.
3. Stabilizacja jednej misji showcase.
4. Mock telemetry dashboard.
5. Review feedback dashboard.
6. Finalniejsze ujęcia pod trailer / store / demo.
7. Build handoff dla Copilota / Claude.

## Bramy jakości
- Smoke build przechodzi.
- Content verify przechodzi.
- Demo mission ma jednoznaczny początek, środek i koniec.
- Retry / next są widoczne i zrozumiałe.
- Telemetry events mają mapę eventów.
- Feedback form ma pola do klasyfikacji problemów.
- Android preview pipeline ma checklistę artefaktów.
