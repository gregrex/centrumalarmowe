
# V5 — playable vertical slice scope

## Cel
Dostarczyć pierwszy grywalny przebieg od ekranu startowego do raportu końca rundy.

## Zakres obowiązkowy
1. Ekran Home.
2. Przycisk Quick Play.
3. Utworzenie lobby demo lub lobby szybkie.
4. Ready-check i automatyczne dociągnięcie BOT-ów.
5. Start sesji.
6. Uruchomienie HUD dla 4 ról.
7. Obsługa 3 pierwszych typów incydentów:
   - incydent medyczny o wysokim priorytecie,
   - pożar mieszkania o średnim priorytecie,
   - zakłócenie porządku / przemoc domowa.
8. Przypisanie jednostek.
9. Aktualizacja statusów incydentów i jednostek.
10. Fallback BOT po rozłączeniu gracza.
11. Raport końca rundy z KPI.

## Zakres celowo odłożony
- pełna kampania,
- sklep / monetyzacja,
- pełna telemetria liveops,
- matchmaking publiczny z rankingiem,
- rozbudowane animacje postaci,
- produkcyjne voice-over,
- pełna ekonomia progresji.

## Kryteria ukończenia
- można wejść do gry w mniej niż 3 kliknięcia,
- pionowy ekran telefonu pozostaje czytelny,
- BOT zastępuje brakujące role w max 2 sekundy,
- pierwsza sesja trwa 5–8 minut,
- raport końca rundy pokazuje min. 8 KPI,
- content jest ładowany z JSON i może być walidowany skryptem.
