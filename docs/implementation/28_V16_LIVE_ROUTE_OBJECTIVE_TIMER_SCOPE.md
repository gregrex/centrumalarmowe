# V16 — Live Route / Objective / Timer Scope

## Cel
Dowieźć jedną **demo-misję** do poziomu „prawie grywalnego” vertical slice, w którym:
- trasy są widoczne jako żywa warstwa na mapie,
- cele misji przechodzą stany w runtime HUD,
- runda ma prosty zegar i fazy napięcia,
- eskalacje łańcuchowe uderzają w gracza czytelnym pakietem UI/audio/VFX,
- grafika, obiekty, sceny, menu, dźwięki i muzyka tworzą spójny produkt.

## Zakres
1. Live route renderer.
2. Round timer.
3. Objective transition HUD.
4. Chain escalation runtime.
5. Demo polish.

## Nie robimy w tej wersji
- pełnej symulacji ruchu pojazdów,
- prawdziwego pathfindera produkcyjnego,
- kompletnego matchmakingu live-service,
- finalnej oprawy audio z masteringiem.
