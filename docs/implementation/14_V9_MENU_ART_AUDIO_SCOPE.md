# V9 Menu, Art, Audio Scope

## Cel
Dowieźć warstwę prezentacji produktu tak, aby agent kodujący mógł zbudować:
- menu główne,
- flow wejścia do gry,
- sceny i warianty wizualne,
- biblioteki obiektów,
- warstwę muzyki i SFX,
- meta progression,
- prosty theme pack pobierany z backendu.

## Co wchodzi do scope v9
1. Main menu z pięcioma głównymi ścieżkami:
   - Campaign,
   - Quick Play,
   - Coop,
   - Profile,
   - Settings.
2. Scene themes:
   - City Day,
   - City Night,
   - Rain Alert,
   - Storm Response,
   - Winter Freeze,
   - Blackout Emergency.
3. Hero objects:
   - ambulance,
   - fire truck,
   - police van,
   - operator desk,
   - dispatch console,
   - radio wall,
   - city marker tower,
   - mobile command van.
4. Audio routing:
   - menu music,
   - in-session music,
   - UI confirm / cancel / warning,
   - role radio cues,
   - critical escalation layer.
5. Meta progression:
   - account level,
   - role proficiency,
   - cosmetics,
   - scene unlocks,
   - badge rewards.

## Zasady
- nie buduj fotorealizmu; buduj premium stylized realism 2D,
- nie przeładowuj menu; maximum 1 hero focus + 1 supporting panel,
- każda scena musi czytelnie działać na pionowym telefonie,
- audio ma wzmacniać decyzję, nie zasłaniać alarmów,
- content-driven first: scene packs, object packs i audio states mają być ładowane z danych.
