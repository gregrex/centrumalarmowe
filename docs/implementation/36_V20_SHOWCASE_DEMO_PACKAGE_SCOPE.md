# V20 Showcase Demo Package Scope

Cel v20: przygotować materiał wejściowy do jednej pokazowej, pionowej ścieżki produktu, którą agent będzie mógł łatwo domknąć do wersji demo.

## Zakres
- first-time-user onboarding od pierwszego uruchomienia,
- misja showcase z ustalonym rytmem tutorial → runtime → recovery → report,
- flow prezentacyjne do nagrania wideo i zrzutów ekranu,
- player-facing polish pack dla UI, scen i audio,
- checklisty build review dla Copilota/Claude.

## Krytyczne rezultaty
1. Home i onboarding muszą tłumaczyć produkt w mniej niż 90 sekund.
2. Jedna misja showcase musi mieć czytelny początek, środek i finał.
3. Demo musi nadawać się do nagrania wideo bez dodatkowego tłumaczenia od developera.
4. Wszystkie elementy muszą być content-driven i łatwe do podmiany.
