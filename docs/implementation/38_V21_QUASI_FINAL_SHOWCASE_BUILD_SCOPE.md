# V21 — Quasi-final showcase build scope

## Cel
Przygotować pakiet jednej misji showcase tak, aby agent mógł zamknąć pierwszy build testowy do:
- wewnętrznego review,
- nagrania demo,
- prezentacji pionowego slice,
- pierwszych screenshotów store,
- kontrolowanego playtestu.

## Zakres v21
1. Review build checklist.
2. Test build deployment flow.
3. Capture mode z ograniczeniem UI noise.
4. Store shot mocks i capsule framing.
5. Player-facing polish jednego showcase flow.
6. Telemetria review build i feedback loop.
7. Testowe wdrożenie API + klienta do środowiska review.

## Główne bramy jakości
- Home do mission complete działa bez ślepych ekranów.
- Capture mode daje czysty układ pod nagranie 9:16 i 16:9.
- Retry/Next flow ma zrozumiałe CTA.
- Review build zawiera checklistę blokującą: brak placeholderów krytycznych, brak pustych ekranów, brak martwych przycisków.
- Demo misja ma spójność audio, grafiki, pacingu i raportu końcowego.
