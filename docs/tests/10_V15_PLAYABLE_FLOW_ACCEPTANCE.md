# V15 Playable Flow Acceptance

## Test przejścia
1. Użytkownik wchodzi do Home.
2. Otwiera misję kampanii.
3. Przechodzi briefing i readiness.
4. W runtime map dispatchuje jednostkę do incydentu.
5. Co najmniej jeden objective przechodzi na progress/completed.
6. Jeden incydent eskaluje i podnosi city pressure.
7. Gracz kończy misję i widzi report progression.
8. Retry flow działa.
