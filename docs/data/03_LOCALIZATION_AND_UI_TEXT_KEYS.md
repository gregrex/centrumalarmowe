# Localization and UI text keys

## Cel
Teksty muszą być oddzielone od logiki, żeby łatwo zrobić PL/EN i późniejsze inne języki.

## Zbiory tekstów
- menu,
- onboarding,
- alerty,
- tutorial,
- role,
- opisy incydentów,
- statusy jednostek,
- komunikaty botów,
- błędy sieciowe,
- wyniki końca misji.

## Reguły
- klucz jest stabilny,
- treść może mieć parametry,
- brak kopiowania tekstu w prefabach,
- każda nowa funkcja dodaje własne klucze i wpisy fallback.
