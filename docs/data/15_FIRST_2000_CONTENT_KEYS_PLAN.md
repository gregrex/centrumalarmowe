# Plan pierwszych 2000 kluczy contentowych

## Zakres
V8 przesuwa projekt do poziomu, w którym agent powinien utrzymywać jeden globalny plan kluczy:
- słowniki,
- UI text keys,
- scenariusze,
- audio cues,
- art prompt packs,
- round loop configi.

## Cele
1. Unikać chaosu nazewniczego.
2. Ułatwić lokalizację i balans.
3. Ułatwić integrację admina i importu.
4. Ułatwić generowanie kolejnych paczek contentu.

## Segmenty
- 1–300: role, stany, alerty,
- 301–700: incydenty i warianty,
- 701–1000: jednostki, cooldowny, ETA,
- 1001–1300: UI/HUD,
- 1301–1600: audio i music states,
- 1601–2000: mapy, trasy, overlaye, heatmapy.
