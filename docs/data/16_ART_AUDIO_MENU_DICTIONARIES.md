## Zasada ID
Format:
`category.subcategory.name.variant`

Przykłady:
- `scene.menu.city_day.default`
- `hero.vehicle.ambulance.alpha`
- `audio.menu.calm.v1`
- `sfx.ui.confirm.soft`
