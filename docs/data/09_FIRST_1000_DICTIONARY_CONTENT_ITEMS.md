
# First 1000 dictionary / content items plan

## Podział
1. role i sloty — 40
2. typy jednostek — 120
3. statusy jednostek — 60
4. typy incydentów — 180
5. severity i priorytety — 40
6. statusy incydentów — 80
7. komunikaty UI — 180
8. alerty audio i VFX — 80
9. event chains — 120
10. portrety / art prompt ids — 60
11. ikony / atlasy — 40

## Reguły
- każdy wpis ma `id`, `group`, `label`, `sortOrder`,
- wszystkie ID po angielsku i bez spacji,
- wszystkie teksty użytkownika lokalizowane,
- agent ma dopisywać nowe grupy tylko po walidacji.
