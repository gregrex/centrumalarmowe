# Ready Check, Handoff, Rejoin, Post-Round Sync

## Ready check sync
- role slot state,
- player id,
- bot fallback mode,
- ready state,
- network quality.

## Handoff
- player leaves,
- bot takes slot,
- player rejoins,
- state restored if safe.

## Post-round sync
- host computes summary,
- all clients receive same grade packet,
- timeline + rewards consistent,
- retry vote optional.
